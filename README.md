# Project3-Portfolio-Project-Walkthrough-Kaggle-Titanic-


A classic introductory data science challenge that involves building a predictive model to determine which passengers survived the Titanic disaster. Gain experience in data cleaning, feature engineering, and applying ML algorithms to create a classification model.
It will develop skills in Python programming, exploratory data analysis (EDA), and model evaluation using libraries like pandas,numpy, and scikit-learn.
